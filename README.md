# Meditrack - Landing Page

Esta es la landing page para Meditrack, una solución innovadora para el cuidado de adultos mayores.

## Descripción

Meditrack ofrece soluciones innovadoras para el cuidado de adultos mayores, combinando tecnología avanzada con un enfoque humano para garantizar su seguridad y tranquilidad. Esta landing page presenta los servicios, características y planes de Meditrack.

## Características

- **Sección Hero:** Presenta el eslogan principal y las llamadas a la acción.
- **Sección de Servicios:** Detalla los problemas del cuidado geriátrico y cómo Meditrack los soluciona.
- **Sección de Impacto:** Muestra estadísticas sobre el impacto positivo de Meditrack.
- **Tecnología:** Describe el dispositivo Meditrack y sus características.
- **Misión y Visión:** Explica la filosofía de la empresa.
- **Planes:** Presenta los diferentes planes de servicio.
- **Contacto:** Un formulario para que los usuarios se pongan en contacto.

## Tecnologías Utilizadas

- HTML5
- CSS3
- Bootstrap 5
- JavaScript

## Cómo ejecutar el proyecto

1. Clona este repositorio.
2. Abre el archivo `index.html` en tu navegador web.
