# Protección de Datos – MediTrack

**Última actualización: Noviembre 11, 2025**

En **MediTrack** (RurasqaySoft), la protección de tus datos personales y de salud es parte fundamental de nuestro compromiso contigo. Este documento describe las medidas técnicas y organizativas implementadas para garantizar la seguridad de tu información.

## 1. Principios de Protección de Datos

Los datos que recopilamos son tratados conforme a los siguientes principios:

### Legalidad y Transparencia
- Recopilamos datos únicamente con tu consentimiento informado
- Te informamos claramente sobre qué datos recopilamos y cómo los usamos
- Cumplimos con todas las regulaciones aplicables (LOPD, GDPR, HIPAA)

### Minimización de Datos
- Solo recopilamos datos estrictamente necesarios para el servicio
- No solicitamos información innecesaria o excesiva
- Evaluamos regularmente la necesidad de cada dato recopilado

### Exactitud
- Mantenemos tus datos actualizados y precisos
- Facilitamos herramientas para que corrijas información incorrecta
- Verificamos periódicamente la calidad de los datos almacenados

### Limitación de Finalidad
- Utilizamos tus datos únicamente para los fines declarados
- No reutilizamos datos para propósitos incompatibles sin tu consentimiento
- Cada uso de datos está documentado y justificado

### Integridad y Confidencialidad
- Protegemos tus datos contra accesos no autorizados
- Prevenimos pérdidas, alteraciones o divulgaciones accidentales
- Mantenemos la confidencialidad en todo momento

## 2. Medidas Técnicas de Seguridad

### Cifrado
- **Cifrado AES-256** para datos en reposo
- **TLS 1.3** para datos en tránsito
- **Cifrado de extremo a extremo** para comunicaciones sensibles
- **Hashing seguro** (bcrypt) para contraseñas

### Control de Acceso
- **Autenticación multifactor (MFA)** obligatoria para cuentas
- **Control de acceso basado en roles (RBAC)**
- **Principio de menor privilegio** aplicado a todos los usuarios
- **Tokens de sesión** con expiración automática
- **Registro de auditoría** de todos los accesos a datos sensibles

### Infraestructura Segura
- **Servidores certificados** en centros de datos seguros
- **Firewalls** y sistemas de detección de intrusiones
- **Backups automáticos** cifrados y redundantes
- **Monitoreo 24/7** de seguridad y anomalías
- **Actualizaciones de seguridad** aplicadas regularmente

### Dispositivos IoT
- **Autenticación única** por dispositivo
- **Comunicación cifrada** con la plataforma
- **Firmware actualizable** de forma segura
- **Certificados digitales** para verificación de dispositivos

## 3. Medidas Organizativas

### Políticas Internas
- Política de seguridad de la información documentada
- Procedimientos de respuesta ante incidentes de seguridad
- Protocolo de notificación de brechas de seguridad
- Plan de continuidad de negocio y recuperación de desastres

### Capacitación del Personal
- Formación obligatoria en protección de datos y privacidad
- Actualizaciones periódicas sobre amenazas de seguridad
- Concienciación sobre ingeniería social y phishing
- Evaluaciones de competencia en seguridad

### Acuerdos de Confidencialidad
- Todo el personal firma acuerdos de confidencialidad (NDA)
- Proveedores externos sujetos a acuerdos de procesamiento de datos
- Auditorías de cumplimiento a terceros

### Gestión de Proveedores
- Evaluación rigurosa de seguridad de proveedores
- Contratos con cláusulas de protección de datos
- Auditorías periódicas de cumplimiento
- Limitación de acceso a datos según necesidad

## 4. Protección Específica de Datos Médicos

### Cumplimiento Normativo
- **HIPAA** (Health Insurance Portability and Accountability Act)
- **Ley de Protección de Datos Personales** de Perú
- **GDPR** para usuarios europeos
- Normativas locales de salud aplicables

### Datos de Salud
- Clasificados como **categoría especial de datos**
- Consentimiento explícito requerido para su procesamiento
- Acceso restringido solo a personal autorizado
- Segregación de datos médicos de otros datos

### Registros de Auditoría
- Logs completos de acceso a datos de salud
- Trazabilidad de todas las operaciones
- Revisión periódica de accesos anómalos
- Conservación segura de registros de auditoría

## 5. Tus Derechos de Protección de Datos

Puedes ejercer los siguientes derechos en cualquier momento:

### Derecho de Acceso
- Obtener copia de todos tus datos personales
- Conocer cómo se procesan tus datos
- Acceder a registros de quién ha accedido a tu información

### Derecho de Rectificación
- Corregir datos inexactos o incompletos
- Actualizar información desactualizada

### Derecho de Supresión ("Derecho al Olvido")
- Solicitar la eliminación de tus datos
- Revocación de consentimientos otorgados

### Derecho de Portabilidad
- Exportar tus datos en formato estructurado
- Transferir tus datos a otro proveedor

### Derecho de Oposición
- Oponerte al procesamiento de tus datos
- Limitar ciertos usos de tu información

### Derecho a No Ser Objeto de Decisiones Automatizadas
- Solicitar revisión humana de decisiones automatizadas
- Comprender la lógica de algoritmos que te afectan

## 6. Gestión de Incidentes de Seguridad

### Detección y Respuesta
- Monitoreo continuo para detectar brechas de seguridad
- Equipo de respuesta a incidentes disponible 24/7
- Protocolo de contención, erradicación y recuperación

### Notificación
- Notificación a usuarios afectados dentro de 72 horas
- Comunicación a autoridades de protección de datos según ley
- Transparencia sobre incidentes y medidas correctivas

## 7. Transferencia Internacional de Datos

- Datos almacenados principalmente en servidores en Perú
- Transferencias internacionales solo con garantías adecuadas
- Cláusulas contractuales tipo para transferencias fuera de Perú
- Cumplimiento con regulaciones de cada jurisdicción
